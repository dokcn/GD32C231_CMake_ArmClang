# GD32C2x1 firmware library

Download `GD32C2x1_Firmware_Library_V1.3.0` from the GigaDevice website, accept the
SLA, and copy the package `Firmware` directory here.

The project expects this layout:

```text
Firmware/
  CMSIS/
    Core/Include/
    GD/GD32C2x1/
      Include/
      Source/
        ARM/startup_gd32c2x1.s
        system_gd32c2x1.c
  GD32C2x1_standard_peripheral/
    Include/
    Source/
```
